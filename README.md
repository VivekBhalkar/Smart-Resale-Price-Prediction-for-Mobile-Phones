## 📝 Project Description  

The **Smart Resale Price Prediction for Mobile Phones** is a machine learning-based web application developed to estimate the resale value of used smartphones based on multiple influencing factors such as brand, model specifications, age, and physical condition.

The project begins with collecting and preprocessing a dataset, where missing values are handled and categorical data is encoded for machine learning models. Exploratory Data Analysis (EDA) is performed to understand feature relationships and their impact on resale price.

Various machine learning models such as **Linear Regression, Decision Tree, and Random Forest** are implemented and evaluated using performance metrics like **R² score, Mean Absolute Error (MAE), and Root Mean Square Error (RMSE)**. The best-performing model is selected after tuning and optimization.

The final system is deployed as a web application using **Flask**, allowing users to input mobile details and receive instant resale price predictions. This system improves decision-making and enhances transparency in the second-hand mobile market.
